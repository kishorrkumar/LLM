# Step 1: Tokenization Fundamentals

Welcome to the first step of my **LLM Learning Journey**.

In this notebook, I explore one of the most fundamental concepts behind Large Language Models (LLMs): **Tokenization**. Before a model can understand or generate text, it must first convert raw text into smaller units called **tokens**.

## 📖 What I Learned

* Reading text data from a file using Python.
* Understanding the difference between **characters**, **words**, and **tokens**.
* Using Python's `re` module to split text with regular expressions.
* Separating punctuation from words.
* Removing whitespace and empty strings from tokenized output.
* Building a simple rule-based tokenizer using regular expressions.

## 🛠️ Technologies Used

* Python
* Regular Expressions (`re`)
* Jupyter Notebook

## 📂 Files

* `tokenizer_1.ipynb` – Introduction to tokenization using Python and regular expressions.

## 🎯 Learning Outcome

By completing this step, I gained an understanding of how raw text is preprocessed before being fed into Large Language Models. This forms the foundation for more advanced topics such as vocabulary creation, encoding, embeddings, and transformer architectures.
